# SCD30

### Instructions

1. [Enable I2C on the Raspberry Pi](https://learn.adafruit.com/adafruits-raspberry-pi-lesson-4-gpio-setup/configuring-i2c)
2. Build `make build`
3. Run `make run`
4. Clean up `make clean`
